# iPhoneBezel

A SwiftUI Library which provides bezels and a home button to your bezel-less iPhone 

## Installation
Google "Swift Package Manager"



## Usage

```swift
import iPhoneBezel
import SwiftUI

struct ContentView: View {
    var body: some View {
        iPhoneBezel {
            // Add your SwiftUI View here
        }
    }
}

```
